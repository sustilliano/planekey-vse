# PlaneKey IDE Add-on: Enterprise Wedge

## Positioning

PlaneKey is the trust layer for AI-generated software changes.

AI coding assistants create or modify files. PlaneKey answers whether those changes belong, are safe, and match dev/builder/live truth.

## MVP buyer-visible checks

- Secret/public repo risk
- Tracking pixel and email-capture residue
- Agent runtime contamination
- Route graph impact
- Dev/build/live integrity drift
- Safe bundle readiness

## Enterprise ladder

- Free/local: IDE add-on + local pk-client scans
- Pro: history, safe bundles, private indexes
- Team: bridge-service, shared dashboards, alerts
- Enterprise: self-hosted server-core, SSO/SAML, audit exports, policy-as-code, Cloudflare/CI integration
